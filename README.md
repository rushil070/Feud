# Coin Clash
A 2 player competitive browser based game where two players compete to see who can win 20 coins first.

Try It: https://rushil070.github.io/Feud/

Link to Code: https://github.com/rushil070/Feud

- Once opening the game, read the instructions and click close.
- For the best gaming experience, set your browser zoom to 90%.
- WASD and the arrow keys can be used by a two different players to compete.
- Three modes are available.
    - "Start Game" is for the regular game, spawning one coin at a time
    - "Hard Mode" generates 5 coins at once
    - "Fast Mode" increases the character's speed
    - "10 Mode" is a quicker mode where players need to collect 10 coins instead of 20.
